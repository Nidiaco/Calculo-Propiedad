# Calculo-Propiedad

A Spanish-language property investment calculator designed to help co-owners make informed financial decisions.

Built as a lightweight single-page app with no dependencies, no backend, and no build step required.

---

## What it does

Helps two or more people analyse a shared property investment by calculating how income, expenses and tax affect each person differently based on their ownership percentage.

It also compares what that same capital would return if invested in a New Zealand bank term deposit instead, so you can decide what makes more financial sense.

---

## Features

👥 **Co-ownership split** based on individual contributions

💰 **Rental income and expense calculator** with proportional splits per owner

🧾 **Expense breakdown** showing each person's share of rates, insurance, maintenance and other costs

📊 **Summary and ROI analysis** per owner across weekly, monthly and annual periods

🏠 **Property vs Term Deposit comparison** using each owner's actual capital and individual RWT tax rate

🏦 **Exploratory deposit calculator** to test any amount across all major NZ banks and terms

📝 **Agreement notes** section to document shared decisions

---

## NZ Banks included

ANZ, ASB, BNZ, Kiwibank, Westpac, Heartland and Rabobank across 6, 9, 12, 18 and 24 month terms.

Rates referenced from interest.co.nz and moneyhub.co.nz as of May 2026.

---

## Tech stack

Plain HTML, Vanilla JavaScript and Tailwind CSS via CDN.
No frameworks. No npm. No build process.

---
